# wip-vs-leadtime
monte carlo simulation of wip vs leadtime.

This small tool demostrates the relation between wip limits and bottlenecks in a project.

Of the second tabpage of the tool you can play with the parameters of the simulation.
The tool accepts a jason file as its configuration.
You can modify the (swim)lanes, add or remove teammembers, change wip limits etc.

If you have any questions or suggestions please let me know.
